MONIMONI APP AI INTERACTIVE SIMULATOR WORKSHOP
--------------------------------------------------------------------------

本系统不是普通聊天框。它会先阅读剧本，判断应该套用哪一种模拟器类型，再按固定模块推进体验。

用户不需要在一大段设定里寻找可操作内容。选项会变成按钮；角色、联系人、记忆和存档会沉淀下来；每一次推进都回到可追踪的运行台。

S-01

导入剧本

粘贴原文，或从默认模拟器开始试玩。

S-02

解析配置

提取角色、关系、规则、玩法循环和初始选项。

S-03

进入剧场

用按钮推进下一段情节，也可以输入自由行动。

S-04

保存进度

联系人、角色记忆和当前节点都可以下次继续。

SIMULATOR PRESETS

不同剧本，不该使用同一种续写方式。

TYPE-ADATING OBSERVATION

恋综 / 换乘观察室

前任线、匿名短信、观察室推理、约会任务和观众热度共同推进。

TYPE-BRELATIONSHIP LIFE

角色关系生活模拟

固定角色群、长期记忆、好感变化、秘密累积和日常事件。

TYPE-CMYSTERY FILE

悬疑推理 / 案件模拟

线索、证词、时间线和嫌疑人状态被拆成可追踪的调查板。

TYPE-DWORLD SANDBOX

世界观沙盒 / 经营探索

地点、阵营、资源和世界事件构成开放但可记录的长期模拟器。

RUN CONSOLE

每一次生成，都应该被页面理解。
