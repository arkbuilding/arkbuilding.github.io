MONIMONI APP AI INTERACTIVE SIMULATOR WORKSHOP
--------------------------------------------------------------------------

本系统不是普通聊天框。它会先阅读剧本，判断应该套用哪一种模拟器类型，再按固定模块推进体验。

用户不需要在一大段设定里寻找可操作内容。选项会变成按钮；角色、联系人、记忆和存档会沉淀下来；每一次推进都回到可追踪的运行台。

LOCAL LIBRARY

MoniMoni 会像一个模拟器阅读器或本地图书馆，把你玩过的换乘模拟器、嫂嫂/嫂子模拟器、恋综模拟器和其他文字剧本聚合保存。原来散落在各处的 txt、设定和存档，可以在一个地方继续打开、整理和推进。

S-01

导入剧本

粘贴 txt 原文，导入换乘模拟器、嫂嫂/嫂子模拟器、恋综模拟器或其他文字剧本。

S-02

解析配置

提取角色、关系、规则、玩法循环和初始选项。

S-03

进入剧场

用按钮推进下一段情节，也可以输入自由行动。

S-04

保存进度

联系人、角色记忆、当前节点和玩过的模拟器都可以下次继续。

SIMULATOR PRESETS

不同剧本，不该使用同一种续写方式。

TYPE-ADATING OBSERVATION

恋综 / 换乘观察室

前任线、匿名短信、观察室推理、约会任务和观众热度共同推进。

TYPE-BRELATIONSHIP LIFE

嫂嫂 / 嫂子 / 角色关系生活模拟

适合嫂嫂模拟器、嫂子模拟器、韩娱模拟器、固定角色群、长期记忆、好感变化和秘密累积。

TYPE-CMYSTERY FILE

悬疑推理 / 案件模拟

线索、证词、时间线和嫌疑人状态被拆成可追踪的调查板。

TYPE-DWORLD SANDBOX

世界观沙盒 / 经营探索

地点、阵营、资源和世界事件构成开放但可记录的长期模拟器。

RUN CONSOLE

每一次生成，都应该被页面理解。

SESSION · MONI-RUN-032 STATUS · PLAYABLE
MODULE 01 新的情节：重逢小屋 · 入住夜 scene
MODULE 02 对话：角色反应、观察室判断、短信内容 dialogue
MODULE 03 A / B / C / D 四个行动按钮 choices
MODULE 04 联系人记忆与当前存档自动更新 memory

THANK YOU FOR YOUR SCRIPT

让模拟器不再只是一段提示词。

MoniMoni 想把阅读门槛、规则理解、选项整理和进度保存交给系统，让换乘模拟器、嫂嫂/嫂子模拟器和各种文字互动剧本都能被收进同一个本地书架。
